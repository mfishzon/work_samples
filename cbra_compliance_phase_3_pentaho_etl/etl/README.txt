This etl posses was built using Pentaho Kettle 7.1 CE.  

Kettle IDE can be downloaded from here: https://sourceforge.net/projects/pentaho/files/Data%20Integration/

Once the IDE is installed, please import the repository via the IDE Repository Menue.

Browse to cbra_compliance_phase_3 directory and execute the PROCESS TABLES JOB

PROCESS TABLES JOB takes a single parameter generate_config set to false by default.  

Setting the generate_config to true will regenerate flat file -> table mappings.
